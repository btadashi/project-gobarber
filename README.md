## Getting started
### Back-end
```bash
# Clone this repository
$ git clone https://github.com/btadashi/project-gobarber.git

# Install all dependencies:
$ yarn

# Run the migrations:
$ yarn typeorm migration:run

# Start application:
$ yarn dev:server
```
